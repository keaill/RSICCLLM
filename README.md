# RSICCLLM

Official implementation and dataset for the paper:

**RSICCLLM: A Multimodal Large Language Model for Remote Sensing Image Change Captioning**

## News

* The code and dataset will be released soon.
* The paper has been accepted by ECCV 2026.

## Introduction

RSICCLLM is a multimodal large language model for remote sensing image change captioning. Given bi-temporal remote sensing images, the model aims to generate accurate and detailed natural language descriptions of scene changes.

This repository will provide the official implementation, dataset processing scripts, training and inference code, evaluation protocols, and pretrained checkpoints.

## Code and Dataset

The code and dataset are currently being organized and will be made publicly available soon.
